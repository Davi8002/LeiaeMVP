# Leiae MVP

MVP web responsivo em Next.js + React + Tailwind CSS.

## Rodar localmente
1. Instale Node.js 18+.
2. Rode `npm install`.
3. Rode `npm run dev`.
4. Abra `http://localhost:3000`.

## Estrutura
- `components/`
- `pages/`
- `data/`
- `styles/`

## Observacao sobre audio
Os arquivos em `public/audio/*.mp3` foram criados como placeholders para demonstracao de caminho local.
Substitua por audios MP3 reais para reproducao completa durante a apresentacao.
